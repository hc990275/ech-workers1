Usage of ech-workers:
  -dns string
        ECH 查询 DNS 服务器 (default "119.29.29.29:53")
  -ech string
        ECH 查询域名 (default "cloudflare-ech.com")
  -f string
        服务端地址 (格式: x.x.workers.dev:443)
  -ip string
        指定服务端 IP（绕过 DNS 解析）
  -l string
        SOCKS5 监听地址 (格式: [user:pass@]ip:port) (default "127.0.0.1:30000")
  -token string
        身份验证令牌
